Assessment belongs in the manual when students need to understand how evidence of learning is produced.

## Rubrics

A rubric should describe observable work, not only final marks.

### Feedback loop

Feedback can point back to chapters, figures and examples already present in the manual.

## Submission checklist

- Data source described.
- Coordinate reference system checked.
- Map or diagram exported.
- Reflection added.
