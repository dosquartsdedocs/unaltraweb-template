Este manual es una nota de lectura de ejemplo para documentación de proyecto. Muestra cómo un sitio de proyecto puede publicar manuales y material de referencia sin crear un componente específico de lecturas.

El mismo elemento de colección puede contener metadatos bibliográficos, texto de reseña y enlaces mientras la página de biblioteca sigue siendo totalmente dirigida por datos.
