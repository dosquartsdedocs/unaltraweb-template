La majoria d'usuaris només necessita una part petita de Jekyll per mantenir un lloc fill.

| Concepte | Ús en unaltraweb |
|---|---|
| Front matter | Seleccionar layout, idioma, perfils i metadades de navegació |
| Col·leccions | Guardar capítols, resultats, tesis i documentació |
| Includes | Reutilitzar fragments HTML/Liquid del nucli |
| Layouts | Definir famílies de pàgina com perfils, manuals i docs |
| Fitxers de dades | Guardar llistes estructurades sota `_data` |

Davant del dubte, edita contingut i front matter en el lloc fill. Edita el nucli només si el comportament s'ha de compartir entre molts llocs.
