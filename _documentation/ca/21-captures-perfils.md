La plantilla pot generar una revisió visual amb Playwright:

```bash
make screenshots LOCAL_CORE=../unaltraweb
```

L'execució neta escriu PNG a `tmp/render-smoke/`. La documentació copia les captures importants a `assets/img/screenshots/` per mostrar l'aspecte de cada perfil sense iniciar els servidors locals.
