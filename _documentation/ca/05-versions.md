| Pas | Propòsit |
|---|---|
| Actualitzar el checkout del nucli | Provar canvis localment abans de publicar |
| Reconstruir una plantilla filla | Verificar el comportament de perfils des d'un consumidor |
| Etiquetar o publicar la gem | Fer la versió disponible per a llocs fills |
| Actualitzar locks de llocs fills | Moure producció de forma deliberada |

La plantilla és intencionadament lleugera perquè les proves de versió usen la ruta real de consum i no sols el repositori del nucli.
