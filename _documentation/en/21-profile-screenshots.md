The template can generate a visual review set with Playwright:

```bash
make screenshots LOCAL_CORE=../unaltraweb
```

The clean run writes PNG files to `tmp/render-smoke/`. The documentation copies the important captures into `assets/img/screenshots/` so the docs can show what each profile looks like without starting the local servers.
