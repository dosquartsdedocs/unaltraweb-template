`unaltraweb` is a reusable Jekyll core for academic, project, manual and technical documentation sites. A child repository keeps local content and configuration; the core provides layouts, includes, styles, scripts and workflows.

## What belongs in the core

- Shared layouts and includes.
- Site profile behaviour.
- Theme, manual and documentation styles.
- Scripts for metrics and generated assets.
- Reusable GitHub Actions workflows.

## What belongs in the template

- Real content and local configuration.
- Local pages, collections and bibliography.
- Project-specific assets and logos.
- Thin workflow wrappers that call the core.
