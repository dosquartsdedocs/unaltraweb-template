| Paso | Propósito |
|---|---|
| Actualizar el checkout del núcleo | Probar cambios localmente antes de publicar |
| Reconstruir una plantilla hija | Verificar el comportamiento de perfiles desde un consumidor |
| Etiquetar o publicar la gema | Hacer la versión disponible para sitios hijo |
| Actualizar locks de sitios hijo | Mover producción de forma deliberada |

La plantilla es intencionadamente ligera para que las pruebas de versión usen la ruta real de consumo y no solo el repositorio del núcleo.
