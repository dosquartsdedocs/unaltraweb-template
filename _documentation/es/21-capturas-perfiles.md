La plantilla puede generar una revisión visual con Playwright:

```bash
make screenshots LOCAL_CORE=../unaltraweb
```

La ejecución limpia escribe PNG en `tmp/render-smoke/`. La documentación copia las capturas importantes a `assets/img/screenshots/` para mostrar el aspecto de cada perfil sin iniciar los servidores locales.
